# 帝國火星曆

https://martian-imperial-year-table.c4se.jp/

## 略語 (abbreviation)

- grdt : `GregorianDateTime`
- juld : `JulianDay`
- delta_t : `JulianDay.delta_t`
- tert : `TerrestrialTime`
- mrls : Mars Ls (Areocentric Solar Longitude)
- mrsd : `MarsSolDate`
- imsn : `ImperialSolNumber`
- imdt : `ImperialDateTime`
